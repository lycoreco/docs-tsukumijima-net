# 外部プログラム

## 名前

任意の名前を指定します。  
この名前はメニュー項目に表示されます。

## コマンド

実行するプログラムとそのコマンドラインオプションを指定します。  
プログラムはフルパスで指定してください。

指定例:  
"C:\Program Files\EpgTimer_Bon.exe" "%tvpid%"  
C:\Windows\System32\cmd.exe /k echo %start-year%/%start-month%/%start-day%
%start-hour%:%start-minute% %event-name%

## パラメータの挿入

コマンドのキャレット位置に、選択したパラメータを挿入します。
