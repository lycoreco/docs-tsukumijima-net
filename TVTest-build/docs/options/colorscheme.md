# テーマ/配色

テーマで TVTest の見た目を変更することができます。  
左上のコンボボックスからテーマを選択します。

## 保存

テーマを保存します。  
TVTest のあるフォルダの Themes フォルダに設定が保存されます。

## 削除

選択されているテーマを削除します。

## 配色

左のリストで色を変更したい項目を選択して、右のパレットで色を選択します。  
左のリストでは、複数の項目を同時に選択することができます。  
パレットの色は、ダブルクリックして変えることができます(ただし変更の保存はされません)。

## プレビュー

現在の配色をプレビューします。
